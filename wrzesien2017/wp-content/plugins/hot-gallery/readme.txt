== Description ==

Hot Gallery by [HotThemes](http://hot-themes.com/ "HotThemes") shows a set of images taken from a directory on your server in a fancy way. In widget options, you have control over the widget's display. You can change width, height, borders, etc. Widget colors are configurable (background color, font color, tabs color).

== Installation ==

1. Install plugin as usually from WordPress.org, or download, unpack and upload `hot_gallery` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Publish Hot Gallery widget in one of available widget sidebars
4. In the widget options, you can see parameters that defines the widget's look.
5. Make sure that Load jQuery is enabled only if your theme doesn't load it.
6. Make sure that thumbnail images can fit in the widget's width (edit number of thumbnails and their size as % of original image).
7. Enter folder where your images are stored, in example http://www.yoursite.com/images/folder. This will show images only without description text.
8. If you need images with description text, beside images folder, you must list all images in the text field below, like "imagename.jpg||Description text;" (without quotes).
